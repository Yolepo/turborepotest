# Portail Web Modulaire

Ce projet est un portail web modulaire construit avec React, MUI, Turborepo, et pnpm.

## Structure du Projet

- `apps/`: Contient les applications web (portail, admin, docs).
- `modules/`: Contient les modules fonctionnels (abonnés, rc).
- `packages/`: Contient les paquets partagés (theme, api-client, etc.).
- `supabase/`: Contient la configuration et les migrations de la base de données Supabase.
- `tooling/`: Contient les configurations d'outils (ESLint, TypeScript, Prettier).

## Démarrage

1.  Installez les dépendances :
    ```bash
    pnpm install
    ```
2.  Démarrez les applications en mode développement :
    ```bash
    pnpm dev
    ```
